# Refining

[/reference](https://github.com/TiCaLiBrO/Trillia/blob/main/root/refining/reference/sepalinfo.md) and [/practice](https://github.com/TiCaLiBrO/Trillia/blob/main/root/refining/practice/sepalinfo.md)

This branch is the refinery for which your skills will be honed.
You will grow stronger and build real projects to keep as living proof of your progress.
Coming here often means you have a firm understanding of Trillia.

## Reference Branch

The reference branch is a full reference manual for specific keywords, functions, and libraries.


## Practice Branch

The practice branch is where you should go once you have done enough tutorials in the learning branch.
The practice branch contains practice lessons to better solidify your understanding of each concept learned in the tutorials branch. <!-- Like practice lessons in boot.dev -->
It also contains challenges, which are trials that make you create your own algorithms to solve a problem. <!-- Like leetcode or codewars -->
Finally, and most importantly, it contains projects, which give you a project to build, and you're told what parts you need to build, but its up to you to figure out how to build it.
The things that you build are entirely yours to keep.
You can show them off and modify them however you like. <!-- most like projects in boot.dev -->
