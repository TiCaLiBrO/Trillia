# String Assignment

## Prelude

You aren't limited to numbers.
You can also assign strings of text to variables.

## The Task

- Make a variable named `greeting`.

- Use the `=` sign, and give it the value `"Hello"`

- Then, make another line to print the `greeting` variable.

Run the code when you are ready.
