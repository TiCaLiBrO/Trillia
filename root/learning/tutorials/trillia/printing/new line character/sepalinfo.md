There is another way to print a new line.
Using `printn`, you can only create a new line at the end of the text.

You can type `\n` inside of your text to create a new line.
This `\n` is a standard way to make a new line in many programming languages.

### The Task

In this lesson, you will have to use the `print` function, not `printn`.

Using only the `print` function and one line of code, print these three lines of text:

    We
    Eat
    Rocks!

> [!TIP]
> When you use `\n`, you don't need a space between words.
