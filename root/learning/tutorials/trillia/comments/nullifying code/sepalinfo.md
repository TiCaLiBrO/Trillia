# Nullifying Code

## Prelude

Now that you know how to add a comment, it's time to turn functioning code into a comment.

## The Task

You are given the code below:

    printn("I like sharks")
    printn("Me too")

Copy it, and alter it so that the code does not print `Me too`.

> [!TIP]
> Remember, everything to the right of `#` becomes a comment.

> [!IMPORTANT]
>
>     try   sepal_execute
>     when  source_code has 'printn("I like sharks")'
>     and   source_code has 'printn("Me too")'
>     and   standard_output = "I like sharks"
>     catch lesson_passed   = True
